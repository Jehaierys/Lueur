# ✦ Lueur Tarot

> *Lueur* [фр.] — мерцание, сияние, нежный свет

Авторское приложение для таро на Android. Jetpack Compose, Material 3, два уникальных визуальных стиля, полная колода из 78 карт с энергиями на русском, французском и английском.

## Возможности

### Колода
- **78 карт** — Старшие Арканы (0–21) + 4 масти по 14 карт
- Реалистичная **стопка карт** — визуально убывает по мере вытягивания
- **Перемешивание** с анимацией и тактильным откликом
- **Счётчик** оставшихся карт
- **Дно колоды** — кнопка просмотра последней карты (общий фон расклада)
- **Оставшиеся карты** — скрытая функция (⋮), чтобы не мешать workflow

### Расклад
- **Вытягивание по клику** на колоду, анимация появления
- **Перевёрнутые карты** — 30% шанс, настраивается
- **Сетка: 1, 2 или 3 столбца** — переключается прямо в раскладе
- **Группировка по 2 или по 3** — карты группируются визуально
- Кнопка **«Новый расклад»** — сохраняет текущий и очищает поле

### Детальная карточка
- Нажми на любую карту → **bottomSheet** с полным описанием
- Имя на русском, французском, английском
- Ключевые энергии / ключевые слова
- Прямое и перевёрнутое значение
- Большой визуал карты

### Подсказки энергий
- Мини-чипы с ключевыми словами прямо под каждой картой
- **Отключаются в настройках** — для продвинутых тарологов

### История раскладов
- Автосохранение каждого расклада (если включено в настройках)
- Список с превью карт, датой, числом карт
- Разворачиваемый детальный список карт в каждом раскладе
- **Быстрая очистка** одной кнопкой (с подтверждением)
- **Удаление отдельных** раскладов

### Два визуальных стиля

#### 🌊 Lueur (авторский)
Неоновый абстрактный минимализм. Для людей с ADHD и аутическими особенностями восприятия:
- Живые **цвета-ауры** для каждой масти (минт, лаванда, коралл, аква, золото)
- **Анимированные органические блобы** — никакой геометрии, только живые формы
- **Пульсирующий свет** — каждая карта дышит по-своему
- Тёмный фон `#0A0A14` — максимальный контраст без усталости глаз
- Уникальное созвездие точек для каждой карты

#### 📜 Classic (классический)
Золото и пергамент. Традиционная эстетика:
- Тёплые тона `#1A0A08` — глубокий бордо
- Золотые `#D4AF37` рамки и орнаменты
- Угловые завитки, двойная рамка
- Центральный звёздный мотив на рубашке

## Установка и сборка

### Требования
- Android Studio Hedgehog (2023.1.1) или новее
- JDK 17
- Android SDK 35
- minSdk 26 (Android 8.0+)

### Шаги

```bash
# Клонируй репозиторий
git clone https://github.com/yourusername/lueur-tarot.git
cd lueur-tarot

# Открой в Android Studio или собери через CLI
./gradlew assembleDebug

# Установи на устройство
./gradlew installDebug
```

### APK
Debug APK появляется в `app/build/outputs/apk/debug/app-debug.apk`

## Архитектура

```
com.lueur.tarot/
├── data/
│   ├── TarotCard.kt          # Модели данных + все 78 карт с текстами
│   ├── SettingsRepository.kt # DataStore preferences
│   └── HistoryRepository.kt  # JSON-персистентность истории
├── domain/
│   └── TarotViewModel.kt     # Вся логика состояния (SSOT)
└── ui/
    ├── theme/
    │   └── Theme.kt          # Lueur + Classic палитры, Typography
    ├── components/
    │   └── TarotCardViews.kt # Canvas-рендер карт, анимации
    ├── screens/
    │   ├── MainScreen.kt     # Колода, расклад, управление сеткой
    │   ├── CardDetailSheet.kt# Детальная карточка (BottomSheet)
    │   ├── HistoryScreen.kt  # История раскладов
    │   ├── SettingsScreen.kt # Настройки
    │   └── RemainingDeckSheet.kt # Оставшиеся карты
    └── TarotApp.kt           # Навигация, оверлеи
```

**Стек:**
- Jetpack Compose + Material 3
- Navigation Compose
- ViewModel + StateFlow
- DataStore Preferences (настройки)
- Gson (история → JSON)
- Canvas API (кастомный рендер карт)

## Принципы дизайна Lueur

1. **Органичность** — никаких прямых углов в арте, только живые формы
2. **Аура масти** — каждая масть светится своим цветом
3. **Дыхание** — все карты анимированы, ни одна не статична
4. **ADHD-friendly** — яркие маркеры, чёткая иерархия, ничего лишнего
5. **Уважение к практике** — подсказки есть, но отключаются одной кнопкой
